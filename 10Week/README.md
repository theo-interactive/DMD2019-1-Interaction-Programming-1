## 인터랙션 사이트 기획서

기말 프로젝트로 진행할 인터랙션 사이트 기획서 작성합니다.

### 주제 (택 1)
1. 영화 티저 (자신이 좋아하는 영화 소개)
2. 브랜드 프로모션 (브랜드, 제품 소개)
3. 이외의 자유 주제

- 프로젝트 명칭

- 톤앤매너, 컨셉 설정
- 사이트 정보구조

- 사이트 디자인
